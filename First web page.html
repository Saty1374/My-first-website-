1. Web Dev is basically teach us how to make websites in real world and make it functionable. There are two types of Web Developers that are

  • Frontend Dev
  • Backend Dev

In frontend we learn to design and place the buttons as the first look of the website, there can be images of objects and many to look that website intresting. We learn some of the tools for that work like

  • HTML
  • CSS
  • JAVASCRIPT

In Backend services we learn to build the code that allows the data base and an application to communicate with one another. Some tools we use inside this are

  • Node.js
  • React.js
  • MongoDB as database
  • Express as framework
